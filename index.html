🛒 Módulo Carrito - Implementación Final

class Carrito: """Maneja el carrito de compras con cálculos automáticos.""" def __init__(self, usuario: 'Usuario'): self.usuario = usuario self.items: List[CarritoItem] = [] self.fecha_creacion = datetime.now() def agregar_producto(self, producto: 'Producto', cantidad: int = 1): """Agrega un producto al carrito o incrementa cantidad existente.""" if not producto.tiene_stock(cantidad): raise ValueError(f"Stock insuficiente para {producto.nombre}") # Buscar si el producto ya existe for item in self.items: if item.producto.id == producto.id: nueva_cantidad = item.cantidad + cantidad if producto.tiene_stock(nueva_cantidad): item.cantidad = nueva_cantidad item.subtotal = item.precio_unitario * nueva_cantidad self._recalcular_descuentos() return # Crear nuevo item si no existe nuevo_item = CarritoItem(producto, cantidad) self.items.append(nuevo_item) self._recalcular_descuentos()

🏗️ Arquitectura y Decisiones Técnicas

📋 Architecture Decision Records (ADRs)

🗄️ PostgreSQL como DB Principal

Contexto: Necesidad ACID para inventario
Decisión: PostgreSQL vs MySQL/SQLite
Razón: Transacciones robustas + Performance
Resultado: 0 inconsistencias de datos

⚡ Redis para Caching

Contexto: Consultas de menú repetitivas
Decisión: Redis como cache layer
Razón: Reducir latencia <500ms
Resultado: 87 Resultado: 87.3% hit ratio, 72% mejora performance

🏢 Monolito vs Microservicios

Contexto: Equipo pequeño, MVP rápido
Decisión: Monolito modular
Razón: Simplicidad + Deploy fácil
Resultado: 4min deploy time, debugging simple

🔐 JWT para Autenticación
